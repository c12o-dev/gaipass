# gaipass

生成AIパスポート試験対策の個人用 PWA。スマホ中心、UI/可視化重視。

## ゴール

1. **2026年8月の生成AIパスポート試験に合格する**（一次ゴール）
2. 副産物として開発過程を Zenn / note 記事化

## 方針

- 個人用ツール（市場プロダクトではない）
- 運用コストゼロ（完全クライアントサイド、静的配信）
- 問題 200問、シラバス詳細キーワード網羅を優先、量は追わない
- UI / 可視化は楽しみの中核：草グラフ風カレンダー、章別ヒートマップ、合格可能性ゲージ

## 機能（予定）

**合格に効く系**
- [ ] シラバス章別／キーワード別の問題演習（4択MCQ、即時採点＋解説）
- [ ] 間違えた問題の再出題（簡易SRS、3階層キュー）
- [ ] 模擬試験モード（60問60分）
- [ ] 章別カバレッジ表示（チェックリスト）

**UI/可視化の楽しみ系**
- [ ] 草グラフ風カレンダー（日別学習量ヒートマップ）
- [ ] 章別ヒートマップ（5章×キーワード）
- [ ] 合格可能性ゲージ
- [ ] 試験日カウントダウン（トップ常駐）
- [ ] 正解時マイクロインタラクション（Framer Motion）
- [ ] ダークモード

## 技術スタック（予定、着手時に確定）

| 領域 | 候補 |
|---|---|
| フレームワーク | Vite + React + TypeScript |
| スタイル | Tailwind CSS + shadcn/ui |
| アニメーション | Framer Motion |
| 可視化 | Recharts or Visx |
| データ永続化 | IndexedDB (Dexie.js) |
| PWA | vite-plugin-pwa |
| ホスティング | Cloudflare Pages |
