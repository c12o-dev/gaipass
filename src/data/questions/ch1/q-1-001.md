# 問題

「人工知能（Artificial Intelligence）」という言葉が初めて学術的に使用されたとされる会議はどれか。

# 選択肢

- ダートマス会議
- チューリング会議
- チョムスキー会議
- マッカーシー会議

# 解説

1956年にアメリカ・ダートマス大学で開催された「ダートマス会議」で、ジョン・マッカーシーらによって "Artificial Intelligence" という用語が初めて提案・使用された。これをもってAI研究の出発点とされる。
